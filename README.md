# vanilla-blackjack-21
Juego de Blackjack en JavaScript, HTML y CSS. Implementa lógica de cartas, puntajes y mensajes de resultado. Interfaz simple y responsiva.

# 🎲 Vanilla Blackjack

Un juego clásico de **Blackjack (21)** desarrollado con **HTML, CSS y JavaScript puro**.  
Juega contra la computadora, intenta acercarte a 21 puntos sin pasarte y demuestra tu suerte.  

---

## 📌 Características
- Cartas generadas aleatoriamente.
- Sistema de puntajes simple.
- Interfaz responsiva y minimalista.
- Mensajes finales claros y en la parte inferior.
- Código limpio y comentado en inglés.

---

## 🚀 Cómo jugar
1. **Hit** para pedir una carta.
2. **Stand** para plantarte y dejar que la computadora juegue.
3. El que esté más cerca de 21 sin pasarse, gana.

---

## 🛠 Tecnologías usadas
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
